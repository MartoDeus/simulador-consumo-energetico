## Objetivo
Construir un simulador web de energia solar para una casa que permita:
- calcular consumo de electrodomesticos,
- estimar generacion de paneles solares,
- estimar almacenamiento en baterias,
- comparar comportamiento de dia y de noche,
- mostrar resumenes, alertas y consejos.

## Lo que ya esta hecho
- Estructura base de la pagina.
- Selector `Dia / Noche`.
- Panel izquierdo para electrodomesticos.
- Panel izquierdo para baterias.
- Panel central con casa solar como referencia visual.
- Panel derecho con resumen en tiempo real.
- Calculos base de consumo, generacion, almacenamiento, desperdicio y deficit.
- Recomendaciones base de paneles y baterias.
